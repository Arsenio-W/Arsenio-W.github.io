Mijn projecten


Hoger of lager was de eerste project waar we moesten programmeren met een doel. Dit deed ik samen met Reshad, we gebruikte math random om de dobblestenen te rollen.

Escaperoom was de tweede project waar ik aan werkte. De thema was overval ontsnappning als bank berover. Hier moest jouw mede berover een code geven aan de kluis maar soms zeg hij de verkeerde nummer.

Build a game is mijn derde en intressanter project na dobblestenen en escaperoom. Hier moest ik zelf een functionele battleships in grid maken.Ik werkte wel met Duncan en Lucas maar hun moesten hun eigen games maken. Ik gebruikte variablen om de buurten te maken en arrays voor als een schip geraakt werd of geplaatst. Het was heel primitief maar werkte wel.

Revenge of javascript is mijn vierde project waar ik solo werkte aan mijn project. Iedereen mocht zelf bepalen wat ze konden maken. Ik koos voor het spel Bommerman helaas kreeg ik niet de kans om het project volledig te maken waardoor het alleen een concept bleef.

Webcatalogus is mijn vijfde en is mijn huidige project waar ik aan werkte. Ik werkte samen met Reshad en Romir. Er werd Node.js gebruikt om de cards in de subcategories te laten werken.

In deze project moesten we een functionele dashboard maken waar je gevens van een database of api moest op halen en vervolgens het te displayen met Chart.js. ik ben er nog bezig met het netjes maken maar het werkt wel mijn dashboard gaat over het weer!
